🔑 YouTube Data API 키 설정

YouTube Data API를 사용하여 더 많은 검색 결과를 얻고 고급 기능을 활용할 수 있습니다.

🛡️ 개인정보 보호 및 보안
  • 암호화 저장: API 키는 암호화되어 안전하게 저장됩니다
  • 세션 모드: 브라우저를 닫으면 자동으로 삭제됩니다
  • 로컬 처리: 서버에 API 키가 저장되지 않습니다
  • 개인 사용: 본인의 Google Cloud Console API 키만 사용하세요
⚠️ 주의: 공용 컴퓨터에서는 세션 스토리지 모드를 사용하고, 사용 후 반드시 API 키를 삭제하세요.

📖 API 키 발급 방법

  1. Google Cloud Console에 접속
  2. 프로젝트 생성 또는 선택
  3. "API 및 서비스" → "라이브러리"에서 "YouTube Data API v3" 활성화
  4. "사용자 인증 정보"에서 "API 키" 생성
  5. 생성된 API 키를 복사하여 위에 입력
💡 팁: API 키는 "AIza"로 시작하는 39자리 문자열입니다.
