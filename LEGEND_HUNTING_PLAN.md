# 🏆 YouTube 레전드 헌팅 시스템 개발 계획서

## 📋 프로젝트 개요

**목표**: 단일 키워드 검색에서 **다중 키워드 검색을 통한 레전드 유튜브 동영상 발굴** 기능으로 업그레이드

**핵심 컨셉**: 여러 키워드로 동시 검색하여 **레전드 점수**를 기반으로 숨어있는 고품질 동영상을 찾아내는 시스템

---

## 🎯 레전드 헌팅 시스템 핵심 요소

### 1. **레전드 점수 계산 공식**
```javascript
레전드 점수 = (조회수 / 경과개월수) × 구독자 가중치

구독자 가중치 = {
  1만 미만: 5.0
  1만~10만: 3.0  
  10만~100만: 2.0
  100만 이상: 1.0
}

레전드 티어 = {
  3000+ : 슈퍼레전드
  1000+ : 레전드  
  300+  : 준레전드
  기타   : 일반
}
```

### 2. **다중 키워드 검색 로직**
- **검색 방식**: 쉼표(,)로 구분된 키워드 개별 검색
- **결과 통합**: 각 키워드별 상위 10개씩 수집
- **필터링**: 레전드 점수 최소값(기본 100점) 이상만 표시
- **정렬**: 키워드별 묶음 + 레전드 점수 내림차순

## 🎮 사용자 경험 흐름

### **1. 단일 검색 (기존 방식 유지)**
```
검색어 입력 → 하이브리드 검색 → 결과 표시
예시: "요리" 검색 → 관련 동영상들 표시
```

### **2. 다중 검색 (새로운 레전드 헌팅)**
```
다중 키워드 입력 → 키워드별 검색 → 레전드 점수 계산 → 필터링 → 통합 결과 표시
예시: "요리, 레시피, 간단" 검색 → 각 키워드별 레전드 동영상 발굴
```

### **3. 검색 결과 화면 구성**
- **정렬 방식**: 키워드별 그룹핑 + 레전드 점수 내림차순
- **표시 정보**: 기존 정보 + 레전드 점수 + 검색 키워드  
- **필터링**: 레전드 점수 최소값으로 품질 관리
